    scala-thumbnailer

    A library for generating thumbnails from various file types. The API uses streams rather than files.

    Supports the following file types:

    • PDF
    • PNG, JPEG, GIF, BMP
    • DOCX
    • Plain text files

    SBT

    libraryDependencies += "nz.co.rossphillips" % "scala-thumbnailer" % "0.4.0"

    Maven

    <dependency>
    <groupId>nz.co.rossphillips</groupId>
    <artifactId>scala-thumbnailer</artifactId>
    <version>0.4.0</version>
</dependency>
